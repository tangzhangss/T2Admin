# TZCC-REN-ADMIN SYSTEM 


### 介绍


后台管理系统开发框架，页面搭建基于element-template，后端采用springboot多模块，技术选型为springboot-jpa,前后端完全分离

vue3.0

#### 系统介绍  


[T2Admin 管理系统框架介绍](https://blog.csdn.net/qq_34813134/article/details/126674977)
